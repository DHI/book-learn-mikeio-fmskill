Change depth¶

Use the method set_z()

msh = Mesh(fn)
msh.plot();

zn = msh.zn            # nodes, not elements!
zn[zn<-15] = -15.0     # clip depth at 15m
msh.zn = zn

msh.plot();

msh.write('Fake_lake_clip15.mesh')   # save to a new file

See the MIKE IO Mesh Example notebook for more Mesh operations (including shapely operations).
